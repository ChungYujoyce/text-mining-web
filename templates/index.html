Description

Sentiment Analysis

We use sentiment analysis API to detect the sentiment of texts. Users can type words or a short paragraph in an article, we will provide users the possible attitude.

Word Cloud

User can input texts and choose the texts related genre, we will use text classificaiotn method to detect the correct genre, then display a word cloud and correspondng background image.

Fake News Detection

We use macine learning methods, which are SVM, SGD, Logistic regression and so on, to build models to detect whether the news is authentical. User can input a short news, we will provide a predicted label and a number of probability of the truth.
